# StudySage: Intelligent Learning Assistant

## Overview
Mobile learning assistant that evolves with you as you dive into any subject or curriculum.

## Core Features
StudySage ingests your source material, syllabi, and learning objectives to create deeply personalized learning journeys.

## Educational Range
- Middle school through medical school support
- Personalized learning paths
- Adaptive curriculum mapping
